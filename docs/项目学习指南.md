# Codivio 项目学习指南

## 🎯 项目概述

**Codivio** 是一个专为学习分布式系统架构设计的实时代码协作平台，通过实际项目来掌握企业级微服务开发技能。

### 项目特点
- 🏗️ **微服务架构** - 4个服务，职责清晰，便于学习
- ⚡ **实时协作** - WebSocket + 操作转换算法
- 🐳 **容器化部署** - Docker + Docker Compose 一键启动
- 🔐 **企业级安全** - JWT + RBAC 权限控制
- 📊 **可观测性** - 监控、日志、健康检查
